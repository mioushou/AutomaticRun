## [1.0.0] - 2026-04-24

### Added

- Toggle auto-run with a configurable keybind (default: `Left Control` / `Left Stick`)
- Collision detection option to automatically stop when hitting a wall or obstacle
- Controller support (left thumbstick, DPad)
- Keyboard movement keys support (respects player's configured bindings)
- HUD notifications when auto-run is toggled on/off
- Generic Mod Config Menu integration for in-game configuration
- Internationalization support (English and French)
- README and LICENSE
